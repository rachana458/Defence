What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It uses element (tags) to describe content structure.

Headings

Use <h1>..</h1> to <h6> for headings. Example:

<h1>Main title</h1>
<h2>Section</h2>
<h3>Subsection</h3>

Best practice: Use one <h1> per page for the main title, then descend in order.

Paragraphs

The <p> element defines a paragraph. Example: 

<p>This is a paragraph of text.</p>

Anchor (Link) Tag

Links are created with <a href="URL">link text</a>. Example:

Visit example.com
