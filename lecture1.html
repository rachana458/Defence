Lecture 1 - Introduction to Web Development & Internet Working

What is the Web?

The Web is a system of interlinked hypertext documents accessed via the Internet using HTTP/HTTPS. Web pages are written in HTML and often accompanied by CSS and JavaScript.

Client & Server

  • Client: Your browser (Chrome, Firefox) that requests resources.
  • Server: The machine that responds with HTML, images, or data.
  • Protocol: HTTP/HTTPS are rules for communication between client and server.

What is MERN Stack

The MERN stack is a popular collection of tools used to build complete web applications, from the part you see in your browser (frontend) to the part that handles data on a server (backend). The acronym MERN stands for:

  • M-MongoDB
  • E-Express
  • R-React
  • N-Node
